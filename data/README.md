# Data

## Description

'data\diluent_vol_conductivity_data.csv': The ionic conductivity data measured by conductivity meter in electrolytes with four diluents. 
This file can be used to reproduce the diluent content-ionic conductivity figures (Figure 1 and 2).

'data\diluent_model_R2_data.csv': The R^2 results summarized from 'results\diluent_vol_conductivity_*_fit.csv'. 
This file can be used to reproduce Figure 3A. 

'data\diluent_model_RMSE_data.csv': The RMSE results summarized from 'results\diluent_vol_conductivity_*_fit.csv'. 
This file can be used to reproduce Figure 3B. 

'data\diluent_feature.csv': The diluents' features which can be used for feature importance and SHAP analysis.
This file can be used to reproduce Figure 4 and generate Figure 5A, 5B and 6.
